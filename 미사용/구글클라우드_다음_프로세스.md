# 구글 클라우드 접속 후 다음 프로세스

구글 클라우드(Cloud Console)에 접속한 상태에서, 아래 순서대로 진행하면 됩니다.

---

## Step 1. 프로젝트 선택 또는 생성

1. **화면 상단**의 프로젝트 이름(또는 "프로젝트 선택")을 클릭한다.
2. **"새 프로젝트"**를 클릭한다.
3. **프로젝트 이름**에 예: `dead-landing-check` 입력 → **만들기** 클릭.
4. 만들어진 프로젝트가 자동 선택되면 그대로 두고, 아니면 목록에서 **방금 만든 프로젝트**를 선택한다.

→ 이제 **해당 프로젝트가 선택된 상태**에서 아래 단계를 진행한다.

---

## Step 2. Google Sheets API 사용 설정

1. 왼쪽 **☰ (햄버거 메뉴)** 클릭.
2. **API 및 서비스** → **라이브러리** 로 이동.
3. 검색창에 **Google Sheets API** 입력 후 **Google Sheets API** 카드를 클릭.
4. **사용** 버튼을 클릭한다.
5. (선택) 검색창에 **Google Drive API** 입력 후 **사용** 해 두면 나중에 편하다.

→ **Google Sheets API**가 "사용 설정됨" 상태가 되면 다음 단계로.

---

## Step 3. 서비스 계정 만들기

1. 왼쪽 **☰** → **API 및 서비스** → **사용자 인증 정보** 로 이동.
2. **+ 사용자 인증 정보 만들기** 클릭.
3. **서비스 계정** 을 선택한다.
4. **서비스 계정 세부정보**  
   - **서비스 계정 이름**: 예) `dead-landing-bot`  
   - **만들기 및 계속** 클릭.
5. **역할 선택(선택사항)**  
   - 비워두거나 **편집자** 등 선택 후 **계속** 클릭.
6. **완료** 클릭.

→ 서비스 계정 목록에 방금 만든 계정이 보이면 성공.

---

## Step 4. 서비스 계정 키(JSON) 다운로드

1. **사용자 인증 정보** 페이지에서, 방금 만든 **서비스 계정** 행의 **이메일**(예: `dead-landing-bot@...`)을 클릭한다.
2. **키** 탭을 클릭한다.
3. **키 추가** → **새 키 만들기** 클릭.
4. **키 유형**: **JSON** 선택 → **만들기** 클릭.
5. JSON 파일이 PC에 **다운로드**된다.  
   - 파일명 예: `프로젝트이름-xxxxx.json`  
   - 이 파일을 **안전한 폴더**에 두고, **Git/공개 저장소에 올리지 않는다.**

6. JSON 파일을 메모장 등으로 열어 **`client_email`** 값을 확인한다.  
   - 예: `"client_email": "dead-landing-bot@dead-landing-check.iam.gserviceaccount.com"`  
   - 이 **이메일 주소 전체**를 복사해 둔다. (다음 단계에서 스프레드시트 공유할 때 사용)

→ **Step 5**에서 이 이메일로 스프레드시트를 공유하면 된다.

---

## Step 5. 스프레드시트를 서비스 계정과 공유

1. **데드랜딩 솔루션 시트** 구글 스프레드시트를 브라우저에서 연다.
2. 우측 상단 **공유** 버튼을 클릭한다.
3. **사용자 또는 그룹 추가** 칸에 Step 4에서 복사한 **서비스 계정 이메일**을 붙여넣기 한다.
4. 권한을 **편집자**로 선택한다.
5. **전송** (또는 **공유**) 클릭한다.

→ 이제 **API로 해당 스프레드시트를 읽고 쓸 수 있는 상태**가 된다.

---

## Step 6. (코드에서 사용할 때)

- **JSON 키 파일 경로**와 **스프레드시트 ID**를 코드/환경 변수에 넣어 두고,  
  `gspread` + `google-auth` 등으로 인증 후 스프레드시트를 열면 된다.
- 스프레드시트 ID: URL의  
  `https://docs.google.com/spreadsheets/d/` **여기부터** `/edit` **앞까지**  
  예: `1iZuRoTV25gyAREoQaOY8TlYXfPhQXfFYUbzAzTKM9Og`

---

## 체크리스트

| 2 | Google Sheets API 사용 설정 | ☐ |
