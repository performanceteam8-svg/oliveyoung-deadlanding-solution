# 경쟁사 리뷰 분석 솔루션 기획안

## 1. 프로젝트 개요

### 1.1 프로젝트명
**Trustpilot 기반 K-뷰티 경쟁사 리뷰 분석 솔루션**

### 1.2 프로젝트 기간
- 기획 및 설계: 1주
- 개발 및 테스트: 2주
- 운영 및 최적화: 지속적

### 1.3 프로젝트 목적
영국 시장 내 주요 K-뷰티 경쟁사들의 고객 리뷰를 체계적으로 분석하여 소비자 니즈와 인사이트를 도출하고, 이를 올리브영 영국 캠페인의 광고 소재 및 마케팅 전략 수립에 활용

---

## 2. 배경 및 필요성

### 2.1 배경
- 영국 시장 진출을 위한 경쟁사 분석 필요성
- 소비자 피드백 기반 제품/서비스 개선 포인트 파악
- 데이터 기반 마케팅 전략 수립 필요

### 2.2 필요성
- 경쟁사들의 강점/약점 파악을 통한 차별화 전략 수립
- 소비자 니즈 기반 광고 소구점 발굴
- 시장 트렌드 및 고객 만족도 요인 분석

## 3. 분석 대상

## 4. 주요 기능

### 4.1 데이터 수집 기능
- Trustpilot 사이트에서 대상 브랜드 리뷰 자동 크롤링
- 영국 지역 필터링
- 리뷰 텍스트, 평점, 작성일, 작성자 정보 수집
- 정기적 업데이트 (주간/월간)

### 4.2 데이터 분석 기능

#### 4.2.1 키워드 분석
- 리뷰 텍스트에서 자주 등장하는 단어 추출
- 유사 의미 단어 그룹화 및 카테고리화
- 빈도수 기반 내림차순 정렬
- 워드클라우드 시각화

#### 4.2.2 감성 분석
- 긍정/부정/중립 리뷰 분류
- 만족 포인트 및 불만족 포인트 자동 추출
- 평점과 텍스트 감성의 일치도 분석

#### 4.2.3 카테고리별 분석
- **제품 관련**: 품질, 효과, 가격, 다양성
- **서비스 관련**: 배송, 포장, 고객 서비스, 반품/교환
- **브랜드 관련**: 신뢰도, 브랜드 이미지, 추천 의향

### 4.3 인사이트 도출 기능
- 경쟁사별 강점/약점 비교 분석
- 소비자 니즈 우선순위 도출
- 광고 소구점 추천
- 트렌드 변화 추적

### 4.4 리포트 생성 기능
- 주간/월간 분석 리포트 자동 생성
- 시각화 대시보드 제공
- 주요 인사이트 요약 및 제안사항 도출

## 5. 분석 방법론

### 5.1 텍스트 전처리
- 불필요한 특수문자 제거
- 대소문자 통일
- 불용어(Stop Words) 제거
- 어간 추출(Stemming) 또는 표제어 추출(Lemmatization)

### 5.2 키워드 카테고리화 전략
- **의미 기반 클러스터링**: 유사한 의미의 단어를 그룹화
  - 예: "fast", "quick", "rapid" → "빠른 배송"
  - 예: "quality", "excellent", "great" → "제품 품질"
- **도메인 전문 사전 구축**: K-뷰티 및 이커머스 관련 용어 사전
- **TF-IDF 기반 중요 키워드 추출**
- **워드 임베딩 기반 유사도 계산**

### 5.3 감성 분석 방법
- **규칙 기반**: 긍정/부정 사전 활용
- **머신러닝 기반**: 감성 분류 모델 활용
- **평점과 텍스트 교차 검증**

### 5.4 만족/불만족 포인트 추출
- 긍정 리뷰에서 자주 언급되는 키워드 → 만족 포인트
- 부정 리뷰에서 자주 언급되는 키워드 → 불만족 포인트
- 카테고리별 만족도 점수 산출

## 부록: 주요 용어 정의

- **키워드 카테고리화**: 유사한 의미를 가진 단어들을 하나의 카테고리로 묶는 작업
- **감성 분석**: 텍스트의 긍정/부정/중립 감성을 자동으로 판단하는 분석
- **소구점**: 광고에서 소비자에게 어필하는 핵심 메시지 포인트
- **TF-IDF**: 단어의 중요도를 계산하는 통계적 방법
